## File requirements

- **Format:** PNG preferred. SVG also works.
- **Background:** Transparent (no white box)
- **Size:** Minimum 300px wide, ideally 600px+ for retina
- **Colour:** Any — the CSS applies grayscale at low opacity by default,
  and reveals full colour on hover. Both light and dark logos work.

## Where to find them

**Google image search:** `[Brand name] logo PNG transparent background`

**Reliable sources:**
- Wikipedia / Wikimedia Commons (search: `[brand] logo site:commons.wikimedia.org`)
- The brand's official press/media kit page
- Brandfetch.com (free tier)
